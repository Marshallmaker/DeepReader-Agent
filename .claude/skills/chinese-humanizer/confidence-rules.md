# 规则置信度与参数

> 此文件记录每条 AI 痕迹检测规则的置信度、用户的改写力度偏好以及例外条件。
> 每次用户反馈后动态更新。

## 改写力度

```
当前力度：5/10（默认）
范围：
  1-3  → 仅替换最明显的AI套话，不改句式，不注入个性
  4-6  → AI套话 + 句式调整 + 去空洞修饰 + 适度注入个性
  7-9  → 深度改写，段落重组，明显注入个人风格
  10   → 最大改写但不改原意，充分的个性声音
```

## 个性注入参数

```
观点密度：5/10（默认，每300字约1次观点表达）
第一人称偏好：倾向"本文"/"我们"（默认中等频率）
复杂性承认：5/10（默认适度讨论局限性）
节奏变化幅度：5/10（默认中等句长方差）
结论方式：留有余地型（默认）
```
